# 更新日志

本文件记录项目的重要变更。

## [未发布]

### 新增
- 重复任务能力：
- 支持每天 / 每周 / 每月重复
- 支持间隔、结束日期、最多次数
- 完成当前任务后自动生成下一条重复任务

- 子任务清单能力：
- 事件可新增、勾选、删除子任务
- 在事件视图展示子任务完成进度

- 标签系统：
- 事件支持标签录入（逗号分隔）
- 支持按标签筛选
- 视图中展示标签

- 统计面板增强：
- 总事件、进行中、今日完成、逾期数量
- 紧急任务完成率
- 近 7 天完成趋势

### 变更
- 主界面从“单页堆叠”改为“三分区工作台”：
- 总览（统计 + 今日）
- 事件（表单 + 多视图）
- 设置（应用设置 + 提醒设置）

- 设置区移除“缩小到托盘”按钮，统一行为为：
- `_` 最小化进入悬浮球
- `X` 关闭隐藏到托盘

- 悬浮球视觉改为白底黑边实心圆，中心显示未完成紧急事件数

### 修复
- 修复悬浮球拖拽中出现的反向漂移与拖拽异常问题（多轮迭代）
- 修复悬浮球图片原生拖拽导致的拖影问题
- 提升提醒可见性与兜底链路（系统通知 + 托盘气泡 + 应用内横幅）
