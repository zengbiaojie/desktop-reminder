# 更新日志

## [1.1.1] - 2026-04-09

### 新增
- 设置页新增“悬浮窗大小”参数，可在 `32-80` 范围内调整。
- 悬浮球尺寸修改后立即生效，并持久化保存。

### 优化
- 悬浮球默认尺寸调整为更紧凑的大小（默认 `46`）。
- 悬浮球窗口尺寸与渲染层样式联动，避免缩放时样式错位。

## [1.1.0] - 2026-04-09

### 新增
- 日历视图支持月份切换（上个月 / 下个月 / 回到本月）。
- 日历支持点击日期打开“当日详情”弹层查看任务。
- 当日详情支持直接完成、编辑、删除事件。
- 表格视图支持“主行 + 子任务展开行”展示。
- 支持一键双目标打包：安装包 + 便携版。

### 改进
- 全局界面样式重构：配色、间距、圆角、阴影与按钮风格统一。
- 列表/表格/时间轴/日历的层级与可读性优化。
- 子任务交互统一：展开查看、勾选完成、删除。
- 无子任务/无备注时不显示冗余占位文案。
- 紧急/重要/日常标签与危险操作按钮对比度提升。

### 修复
- 修复打包后托盘图标与窗口左上角图标丢失问题。
- 修复打包资源遗漏（`assets` 与 `ddl.png` 未被包含）。
- 修复图标格式兼容问题，统一使用 `assets/app.ico`。
- 运行时图标路径兼容开发态与打包态（含 `process.resourcesPath`）。

## [1.0.0] - 2026-04-07

### 初版功能
- 事件新增、编辑、删除、完成切换与视图切换。
- 优先级与 DDL。
- 多视图（列表 / 日历 / 表格 / 时间轴）。
- 提醒机制（1天 / 1小时 / 10分钟 / 逾期）。
- 托盘驻留与悬浮球。
- 开机自启与置顶设置。
