# Desktop Reminder

一个基于 Electron 的桌面提醒应用，定位是“轻量、常驻、可视化、多粒度任务管理”。

## 功能概览

- 事件管理：新增、编辑、完成切换、删除
- 事件粒度：`urgent` / `important` / `daily`
- DDL 管理：截止时间录入与校验
- 多视图查看：列表 / 日历 / 表格 / 时间轴
- 搜索与筛选：关键词、粒度、状态、标签
- 批量操作：批量完成、批量删除

## 提醒能力

- 提前提醒：1 天 / 1 小时 / 10 分钟
- 逾期提醒
- 提醒设置可持久化
- 多通道提醒兜底：系统通知、托盘气泡、应用内横幅

## 新增功能（当前版本）

- 重复任务
- 支持每天/每周/每月
- 支持间隔、结束日期、最多次数
- 完成当前后自动生成下一条

- 子任务清单
- 支持子任务新增/勾选/删除
- 支持进度展示（例如 `2/5`）

- 标签系统
- 事件标签录入（逗号分隔）
- 标签展示与筛选

- 统计面板
- 总事件、进行中、今日完成、逾期数量
- 紧急任务完成率
- 近 7 天完成趋势

## 桌面行为

- `X` 关闭：隐藏到托盘
- `_` 最小化：切换为悬浮球
- 悬浮球：
- 左键短按恢复主窗口
- 右键菜单（显示主窗口/隐藏悬浮球/退出）
- 中央数字显示“未完成紧急事件数”

## 页面结构

应用已拆分为三大分区，避免单页堆叠：

- 总览：统计面板 + 今日面板
- 事件：事件表单 + 多视图管理
- 设置：应用设置 + 提醒设置

## 安装与运行

```bash
npm install
npm start
```

## 项目结构

```text
src/
  main.js              # 主进程：窗口/托盘/提醒/数据逻辑
  preload.js           # 主窗口预加载 API
  renderer.js          # 主窗口交互与渲染
  index.html           # 主窗口页面
  styles.css           # 主窗口样式
  bubble.html          # 悬浮球窗口
  bubble.css           # 悬浮球样式
  bubble-renderer.js   # 悬浮球交互
  bubble-preload.js    # 悬浮球预加载 API
assets/
  tray.ico
  tray.png
  icon-*.png
```

## 数据存储

应用数据保存在 Electron 的 `userData/events.json`，包含：

- `settings`：应用设置与提醒配置
- `events`：事件数据（包含标签、子任务、重复任务、提醒状态）
