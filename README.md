# Desktop Reminder

基于 Electron 的轻量桌面提醒应用，面向“快速记录 + 到期提醒 + 多视图管理”。

## 核心功能

- 事件管理：新增、编辑、删除、完成/逾期状态切换
- 优先级：`紧急` / `重要` / `日常`
- DDL：支持截止时间、逾期识别、重复任务自动推进
- 多视图：列表 / 表格 / 日历 / 时间轴
- 子任务：支持独立 DDL、顺序调整（上移/下移）、完成/删除
- 标签与备注：支持多标签与筛选

## 新增能力（v1.2.x）

- 总览新增“高风险任务”模块（综合优先级 + DDL 临近 + 逾期评分）
- 总览新增“未来 7 天负载”柱状图
- 子任务 DDL 选择交互修复（可正常打开并选择日期）
- 悬浮球支持“闪烁提醒间隔（秒）”设置（0=关闭）`r`n- 支持“闪烁时屏幕中心文字提示”开关（可选仅闪烁）
- 悬浮球视觉美化与闪烁可视增强（不再放大裁剪）

## 桌面体验

- 窗口置顶（可配置）
- 开机自启（可配置）
- 关闭按钮默认隐藏到托盘（不退出进程）
- 最小化进入可拖拽悬浮球模式
- 悬浮球尺寸可在设置中调整（`32-80`，实时生效）
- 托盘菜单支持恢复主窗口与退出应用
- 托盘图标与窗口左上角图标统一

## 提醒能力

- 支持 DDL 提前 `1天 / 1小时 / 10分钟` 提醒
- 支持逾期提醒
- 支持系统通知 + 应用内提醒日志
- 提醒开关可配置并持久化

## 设置项

- 开机自启
- 窗口置顶
- 提醒开关
- 悬浮窗大小（`32-80`）
- 悬浮球闪烁间隔（秒，`0` 为关闭）`r`n- 闪烁时是否显示屏幕中心文字提示

## 安装与运行

```bash
npm install
npm start
```

## 打包

```bash
npm run build:all
```

或分别打包：

```bash
npm run build          # NSIS 安装包
npm run build:portable # 便携版 exe
```

产物默认位于 `dist/`：

- `Desktop Reminder Setup x.x.x.exe`
- `Desktop Reminder x.x.x.exe`

## 数据存储

数据保存在 Electron `userData/events.json`：

- `settings`：置顶、自启、提醒、悬浮窗大小、悬浮球闪烁间隔等
- `events`：标题、优先级、DDL、标签、子任务、重复规则、状态等
