# Desktop Reminder

基于 Electron 的轻量桌面提醒应用，面向“快速记录 + 到期提醒 + 多视图管理”。

## 核心功能

- 事件管理：新增、编辑、删除、完成/逾期状态切换
- 优先级：`紧急` / `重要` / `日常`
- DDL：支持截止时间、逾期识别、重复任务自动推进
- 多视图：列表 / 表格 / 日历 / 时间轴
- 子任务：支持新增、完成、删除、展开查看
- 标签与备注：支持多标签与筛选

## 子任务能力（最新）

- 子任务支持独立 DDL
- 子任务支持顺序（上移/下移）
- 子任务顺序与 DDL 会持久化保存
- 重复任务生成下一次时，子任务 DDL 会按周期自动顺延

## 总览页能力（最新）

- 统计面板：总数、进行中、今日完成、逾期、完成率
- 今日面板：今天到期 / 已逾期
- 高风险任务：按“优先级 + 临近DDL + 逾期”综合评分展示
- 未来 7 天负载：按天柱状图展示待办量

## 桌面体验

- 窗口置顶（可配置）
- 开机自启（可配置）
- 关闭按钮默认隐藏到托盘（不退出进程）
- 最小化进入可拖拽悬浮球模式
- 托盘菜单支持恢复主窗口与退出应用
- 托盘图标与窗口左上角图标统一

## 提醒能力

- 支持 DDL 提前 `1天 / 1小时 / 10分钟` 提醒
- 支持逾期提醒
- 支持系统通知 + 应用内提醒日志
- 提醒开关可配置并持久化

## 设置项

- 开机自启
- 窗口置顶
- 提醒开关
- 悬浮窗大小（`32-80`，实时生效并持久化）

## 安装与运行

```bash
npm install
npm start
```

## 打包

### 一键打包（安装包 + 便携版）

```bash
npm run build:all
```

### 分别打包

```bash
npm run build          # NSIS 安装包
npm run build:portable # 便携版 exe
```

打包产物默认位于 `dist/`：

- `Desktop Reminder Setup x.x.x.exe`（安装包）
- `Desktop Reminder x.x.x.exe`（便携版）

## 数据存储

数据保存在 Electron `userData/events.json` 中：

- `settings`：置顶、自启、提醒、悬浮窗大小等设置
- `events`：标题、优先级、DDL、标签、子任务、重复规则、状态等

## 图标说明

运行与打包统一优先使用 `assets/app.ico`，并兼容 `tray.ico` / `tray.png` 作为候选。
