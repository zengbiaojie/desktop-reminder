# Desktop Reminder

基于 Electron 的轻量桌面提醒应用，面向“快速记录 + 到期提醒 + 多视图管理”。

## 核心功能

- 事件管理：新增、编辑、删除、完成/逾期状态切换
- 优先级：`紧急` / `重要` / `日常`
- DDL：支持截止时间、逾期识别、重复任务自动推进
- 多视图：列表 / 表格 / 日历 / 时间轴
- 子任务：支持独立 DDL、顺序调整（上移/下移）、完成/删除`r`n- 已修复子任务 DDL 日期选择器交互冲突（现在可正常选择日期）
- 标签与备注：支持多标签与筛选

## 总览页

- 统计面板：总数、进行中、今日完成、逾期、完成率
- 今日面板：今天到期 / 已逾期
- 高风险任务：按“优先级 + 临近DDL + 逾期”综合评分排序
- 未来 7 天负载：按天柱状图展示待办量

## 桌面体验

- 窗口置顶（可配置）
- 开机自启（可配置）
- 关闭按钮默认隐藏到托盘（不退出进程）
- 最小化进入可拖拽悬浮球模式
- 悬浮球尺寸可在设置中调整（`32-80`，实时生效）
- 已修复悬浮球按压/拖拽时异常变大问题
- 托盘菜单支持恢复主窗口与退出应用
- 托盘图标与窗口左上角图标统一

## 提醒能力

- 支持 DDL 提前 `1天 / 1小时 / 10分钟` 提醒
- 支持逾期提醒
- 支持系统通知 + 应用内提醒日志
- 提醒开关可配置并持久化

## 设置项

- 开机自启
- 窗口置顶
- 提醒开关
- 悬浮窗大小（`32-80`）

## 安装与运行

```bash
npm install
npm start
```

## 打包

```bash
npm run build:all
```

或分别打包：

```bash
npm run build          # NSIS 安装包
npm run build:portable # 便携版 exe
```

产物默认位于 `dist/`：

- `Desktop Reminder Setup x.x.x.exe`
- `Desktop Reminder x.x.x.exe`

## 数据存储

数据保存在 Electron `userData/events.json`：

- `settings`：置顶、自启、提醒、悬浮窗大小等
- `events`：标题、优先级、DDL、标签、子任务、重复规则、状态等
