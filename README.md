# Desktop Reminder

基于 Electron 的轻量桌面提醒应用，支持置顶、托盘驻留、悬浮球、事件多视图管理与到期提醒。

## 功能概览

- 事件管理：新增、编辑、删除、完成/逾期状态切换
- 优先级：`紧急` / `重要` / `日常`
- DDL：支持时间设置、逾期识别、重复任务自动推进
- 多视图：列表 / 表格 / 日历 / 时间轴
- 子任务：可添加、展开查看、勾选完成、删除
- 标签与备注：支持多标签与快速筛选

## 桌面体验

- 窗口置顶（可开关）
- 开机自启（可开关）
- 关闭按钮默认隐藏到托盘（不退出进程）
- 最小化进入可拖拽悬浮球模式
- 托盘菜单支持恢复主窗口与退出应用
- 托盘图标与窗口左上角图标统一

## 提醒能力

- 支持 DDL 提前 `1天 / 1小时 / 10分钟` 提醒
- 支持逾期提醒
- 支持系统通知与应用内提醒日志
- 提醒开关可配置并持久化

## 设置项

- 开机自启
- 窗口置顶
- 提醒开关
- 悬浮窗大小（`32-80`，实时生效并持久化）

## 视图说明

### 列表视图
- 卡片形式展示事件信息
- 支持展开子任务并直接勾选/删除
- 已完成与已逾期状态有明显视觉区分

### 表格视图
- 主任务按行展示
- 有子任务时可展开子行（带缩进）
- 子任务可直接完成/删除

### 日历视图
- 支持切换月份与回到本月
- 点击日期可打开“当日详情”弹层
- 当日详情支持完成/编辑/删除

### 时间轴视图
- 按 DDL 时间顺序展示事件
- 标题、元信息与备注分层展示
- 支持展开子任务列表

## 安装与运行

```bash
npm install
npm start
```

## 打包

### 一键打包（安装包 + 便携版）

```bash
npm run build:all
```

### 分别打包

```bash
npm run build          # NSIS 安装包
npm run build:portable # 便携版 exe
```

打包产物位于 `dist/`：

- `Desktop Reminder Setup x.x.x.exe`（安装包）
- `Desktop Reminder x.x.x.exe`（便携版）

## 数据存储

数据保存在 Electron `userData/events.json` 中，包含：

- `settings`：应用设置（置顶、自启、提醒、悬浮窗大小等）
- `events`：事件列表（优先级、DDL、标签、子任务、重复规则、状态等）

## 图标说明

打包与运行时统一使用 `assets/app.ico`，并兼容 `tray.ico` / `tray.png` 作为候选。
